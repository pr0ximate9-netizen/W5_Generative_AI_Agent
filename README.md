# SeatWise Agent Simulation

Multi-agent simulation project for exploring computer lab seat shortage issues.

The project source lives in [`7_generativeAgent/`](./7_generativeAgent/).

## Run

```powershell
cd 7_generativeAgent
pip install -r requirements.txt
streamlit run scripts/app.py
```

For the full Korean project description, see [`7_generativeAgent/README.md`](./7_generativeAgent/README.md).
